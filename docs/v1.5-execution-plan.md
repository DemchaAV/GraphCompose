# GraphCompose v1.5 — Execution Plan

> Companion-документ к [v1.5-audit-and-roadmap.md](./v1.5-audit-and-roadmap.md).
> Это **рабочий tracker**: галки — единица прогресса. Каждая фаза имеет
> подзадачи, файлы, критерии готовности, и явные тесты.
>
> **Условные обозначения**
> - `- [ ]` — открыто
> - `- [x]` — сделано
> - `- [~]` — в работе (опц., не markdown-стандарт, но удобно глазу)
>
> **Размеры**: SP = story point ≈ полдня сосредоточенной работы.
> **Owner**: `@me` пока всё на одном человеке; помечайте, если меняется.
